## ******功能模块******

桃源婚恋交友APP分两大部分，即管理员管理和用户管理。系统按照用户的实际需求开发而来，贴近生活。从管理员出拿到分配好的账号密码可以进入系统，使用相关的系统应用。管理员总体负责整体系统的运行维护，统筹协调。
